My projects

  • Aircraft Analysis Using AWS

    I worked with a bunch of data about airlines, and used a bunch of fancy tools to make sense of it. I started off by using SQL to dig into the features of the data and uploading it onto a cloud platform called AWS using something called PUTTY. Once it was up there, I used some more AWS tools to make a schema for the data that contained huge records.

  • Medical Expense Generation System

    I worked on creating a database for medical services using something called SQL Server. Once the database was set up, I used a tool called MySQL Workbench 8.0 CE to create an application that would make it easier to track data and reduce the need for manual maintenance via Excel.

  • Accenture Data visualization

    I analyzed 1K+ online content data records using Excel & Power BI. After cleaning and processing the data, I identified the top 5 content categories and visualized the results for insights on top performers.

General Information

I have co-authored a research paper titled "Secure Classroom Using AES Algorithm" that was published in the GIS Science Journal, U.G.C. CARE Group-II Journal in May 2021.

From January to May 2021, I led a team of three to develop a secured cloud classroom application using advanced encryption algorithm. The application was designed using C#, DotNet, and Android Studio, and allowed students to retrieve study materials uploaded by faculty. We deployed an accessible database on AWS to upload notes, host quizzes, and track students. The project aimed to provide a secure and efficient platform for students to access educational materials remotely. Our team successfully completed the project and delivered a reliable and user-friendly cloud classroom application.
